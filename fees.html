Fee Structure

At FutureMinds, we keep our fees simple and flexible so families can choose what works best for them. All programs include access to our digital resources, live sessions with qualified tutors, and ongoing support.

Kids Programs (Ages 6-15)

Package Fee Details
Per Session $15 (KES 2,000) Individual class
Monthly Package (8 sessions) $80 (KES 10,500) Works out to $10 per session
Quarterly Package (24 sessions) $210 (KES 27,500) ≈ $8.75 per session
Annual Package (96 sessions) $720 (KES 94,500) ≈ $7.50 per session

Teen & Adult Programs (Ages 16+)

Package Fee Details
Per Session $20 (KES 2,600) Individual class
Monthly Package (8 sessions) $110 (KES 14,500) $13.75 per session
Quarterly Package (24 sessions) $300 (KES 39,500) $12.50 per session
Annual Package (96 sessions) $1,080 (KES 141,500) $11.25 per session

Discounts & Offers

  • Sibling/Family Discount: 10% off for each additional child or parent enrolling.
  • Early Registration: Save 15% when you pay for a full term in advance.
  • Scholarships: Limited opportunities for high-achieving or financially needy students.
